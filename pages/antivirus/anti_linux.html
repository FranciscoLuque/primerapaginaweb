Intalacion de ClamAV para Ubuntu

ClamAV es un software antivirus open source para las plataformas Windows, GNU/Linux, BSD, Solaris, Mac OS, etc..

Los primero sera abrir el terminal de Ubuntu y escribir: sudo apt install clamav

Después volveremos a introducir otro comando: sudo apt install clamtk que se trata de la interfaz gráfica del antivirus
