¿Qué es un software?

Un software es un término informático que hace referencia a un programa o conjunto de programas que mediante una serie de datos, procedimientos y normas nos permiten realizar distintas tareas en un sistema informático.

Tipos de software

  • Software Libre
    • Software Comercial
    • Shareware
  • Software Propietatio
    • Freeware
    • Clipleware
